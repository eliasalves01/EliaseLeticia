Sobre o armazenamento de fotos

As fotos que você adicionar são armazenadas localmente no seu navegador usando localStorage. Isso significa que:

  • As fotos só ficam visíveis neste dispositivo
  • Se você limpar os dados do navegador, as fotos serão perdidas
  • O armazenamento é limitado a aproximadamente 5MB por domínio
  • Fotos muito grandes podem não ser armazenadas corretamente

Para uma solução permanente, seria necessário implementar um sistema de armazenamento em nuvem como Firebase Storage ou AWS S3.
